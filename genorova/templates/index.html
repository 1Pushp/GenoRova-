Platform

Built for Real Drug Discovery

Every component validated against real pharmaceutical benchmarks and Phase 3 clinical trial data.

🧠
VAE Generative Model
Variational Autoencoder with 256-dim latent space trained on ChEMBL. Generates structurally novel, drug-like molecules beyond training distribution.
🔗
AutoDock Vina Docking
Real molecular docking against DHPS, insulin receptor (1IR3), DPP4 (1NNY), and ACE2 (6M0J) target proteins. Binding affinity in kcal/mol.
📊
Phase 3 Clinical Scoring
Composite scoring against real FDA endpoints: HbA1c reduction, fasting glucose, hypoglycemia risk, CV safety. Based on 50 Phase 3 diabetes trials.
🗄️
ChEMBL Integration
Trained on curated ChEMBL datasets for diabetes and infectious disease targets. Novel molecules checked against training data for novelty score.
🔌
REST API Access
Full programmatic access via JSON REST API. Generate, score, and retrieve molecules from any language. Interactive docs at /docs.
📄
HTML Report Export
Download complete molecular analysis reports as self-contained HTML files with embedded structure images, docking results, and ADMET profiles.
Developer API

REST API Documentation

Integrate Genorova AI directly into your research pipeline. Full OpenAPI docs available at /docs

GET /health Service status & model availability 
# Check if Genorova is live
curl https://genorova-api.onrender.com/health

# Response
{
  "status": "running",
  "model": "Genorova AI v1.0",
  "models_loaded": { "diabetes": true, "infection": true },
  "molecules_in_db": 100,
  "best_molecule": "COc1cc2c(cc1OC)C(C)N(S(N)(=O)=O)CC2",
  "best_score": 0.9649
}
POST /generate Generate drug candidates for a disease 
curl -X POST https://genorova-api.onrender.com/generate \
  -H "Content-Type: application/json" \
  -d '{"disease": "diabetes", "count": 10}'

# Response
{
  "disease": "diabetes",
  "count_returned": 10,
  "molecules": [
    {
      "rank": 1,
      "smiles": "COc1cc2c(cc1OC)C(C)N(S(N)(=O)=O)CC2",
      "molecular_weight": 286.35,
      "qed_score": 0.8918,
      "clinical_score": 0.9649,
      "passes_lipinski": true,
      "recommendation": "Strong candidate"
    },
    ...
  ]
}
POST /score Score any SMILES string 
curl -X POST https://genorova-api.onrender.com/score \
  -H "Content-Type: application/json" \
  -d '{"smiles": "COc1cc2c(cc1OC)C(C)N(S(N)(=O)=O)CC2"}'

# Response
{
  "smiles": "COc1cc2c(cc1OC)C(C)N(S(N)(=O)=O)CC2",
  "molecular_weight": 286.35,
  "logp": 0.83,
  "qed_score": 0.8918,
  "sa_score": 2.8319,
  "passes_lipinski": true,
  "clinical_score": 0.9649,
  "recommendation": "Strong candidate"
}
GET /best_molecules Top N molecules ranked by clinical score 
curl https://genorova-api.onrender.com/best_molecules?n=10
