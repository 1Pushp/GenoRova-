How Genorova AI Works

1
Data
Downloads 1,700+ real drug molecules per disease from ChEMBL REST API
2
Train VAE
3-layer Variational Autoencoder (1024→512→256) with cyclic KL annealing
3
Generate
Samples latent space + library screening to produce novel candidates
4
Score
Scores against real Phase 3 clinical trial endpoints (HbA1c, QED, SA, Lipinski)

REST API — Quick Reference

Base URL: http://localhost:8000  |  Interactive docs: http://localhost:8000/docs

GET/health
Service status, model availability, molecule count
curl http://localhost:8000/health
POST/score
Score any SMILES string — returns full ADMET + clinical score
curl -X POST http://localhost:8000/score \ -H "Content-Type: application/json" \ -d '{"smiles": "COc1cc2c(cc1OC)C(C)N(S(N)(=O)=O)CC2"}'
POST/generate
Generate drug candidates — returns ranked list with all properties
curl -X POST http://localhost:8000/generate \ -H "Content-Type: application/json" \ -d '{"disease": "diabetes", "count": 10}'
GET/best_molecules
Top 10 molecules discovered — queries the persistent molecule database
curl http://localhost:8000/best_molecules?n=10
GET/report
Full HTML discovery report with all molecules, scores, and structure images
curl http://localhost:8000/report > report.html

Quick Start

1. Install dependencies

pip install -r requirements.txt conda install -c conda-forge rdkit

2. Download training data

python src/download_data.py

3. Run the full pipeline

python -X utf8 src/run_pipeline.py

4. Start the API server

start_genorova.bat # Windows # or: python -m uvicorn src.api:app --host 0.0.0.0 --port 8000

5. View HTML report

python src/genorova_cli.py report # Opens: outputs/genorova_report.html
